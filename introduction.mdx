Geyser is an open-source proxy that translates the Minecraft: Bedrock Edition network protocol into the Minecraft: Java Edition protocol, enabling cross-platform play without any changes needed on the Java server side.

## What is Geyser?

Geyser acts as middleware between a Bedrock client and a Java server. When a Bedrock player connects, Geyser:

1. Accepts the incoming Bedrock Edition connection on its listener port (default `19132` UDP)
2. Translates all Bedrock packets into their Java equivalents in real time
3. Forwards translated packets to the backend Java server (default port `25565`)
4. Translates Java server responses back to Bedrock format for the client

This means any standard Java Edition server — Vanilla, Paper, Spigot, Fabric — works with Geyser without modifications.

## Key features

- **Protocol translation** — Full bidirectional translation of Bedrock ↔ Java packets including inventory, entities, blocks, and chat
- **Extension API** — Build custom extensions using Geyser's Java API with an event bus, custom commands, and connection access
- **Custom items and blocks** — Register custom items and blocks that appear correctly to Bedrock players
- **Resource pack support** — Serve Bedrock-compatible resource packs to connecting clients
- **Authentication flexibility** — Support for online mode (Microsoft accounts) and offline mode
- **Floodgate integration** — Allow Bedrock players to join without a Java account using the companion [Floodgate](https://github.com/GeyserMC/Floodgate) plugin
